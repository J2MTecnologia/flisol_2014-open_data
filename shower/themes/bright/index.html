Ontologias

Ontologias

  • São construídas a partir de uma terminologia
  • Feitas em XML
  • Geram um OWL (arquivo .xml)
  • Comunica-se com um RDF

O que é OWL, RDF, RDF Schema, XML, XML Schema

Conceitos

  • São arquivos XML com propósito específico

URI

URI

  • URI (Uniform Resource Identifiers) são utilizados para nomear e identificar coisas;
  • Identificação única através de recursos;
  • Recurso: qualquer coisa que tenha um identificador;
  • Podem ser:
    • Relativas
    • Absolutas
